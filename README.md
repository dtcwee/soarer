# Soarer
Flight simulation.
Launch from the [home page](https://mobileflightapp.wordpress.com).
## Soarer (Mobile 📱 )
Use device orientation to fly around the world.
## Soarer PC 🖥 🎮 
Use Keyboard and Gamepad to fly around the world.
## What's New
* Migrated to Cesium 1.42, jQuery 3.7.1, and jQueryUI 1.14.2 (SoarerPC) 
* Static cloud performance improvement.
